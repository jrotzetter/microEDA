# Replace OTU table in a microEDA object

Replace OTU table in a microEDA object

## Usage

``` r
# S4 method for class 'microEDA,otu_table'
otu_table(x) <- value
```

## Arguments

- x:

  A `microEDA` object.

- value:

  An `otu_table` object (or coercible to one).

## Value

The updated `microEDA` object.
