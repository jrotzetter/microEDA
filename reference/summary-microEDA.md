# Summarize a microEDA object

Provides a concise, more detailed overview over a `microEDA` object than
the
[show()](https://jrotzetter.github.io/microEDA/reference/microEDA-class.md)
method.

## Usage

``` r
# S4 method for class 'microEDA'
summary(object, ...)
```

## Arguments

- object:

  A `microEDA` object.

- ...:

  Optional arguments (currently supports `label_width`).

## Value

An invisible list containing summary statistics.
