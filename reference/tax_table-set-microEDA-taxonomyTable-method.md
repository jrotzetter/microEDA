# Replace taxonomic table in a microEDA object

Replace taxonomic table in a microEDA object

## Usage

``` r
# S4 method for class 'microEDA,taxonomyTable'
tax_table(x) <- value
```

## Arguments

- x:

  A `microEDA` object.

- value:

  A `taxonomyTable` object (or coercible to one).

## Value

The updated `microEDA` object.
